# 🚗 Electric Cars in India – Tableau Web App

## 📌 Project

This project is a simple **Flask web application** that displays an **interactive Tableau Story dashboard** about the growth of **Electric Vehicles (EVs) in India**.

The dashboard helps users understand EV trends, adoption, and data insights through visualizations.

---

## 🛠️ Technologies Used

* Python
* Flask
* HTML
* Tableau Public

## ⚙️ How to Run the Project

1. Install Flask

```bash
pip install flask
```

2. Run the application

```bash
python app.py
```

3. Open in browser

```
http://127.0.0.1:5000/
```
